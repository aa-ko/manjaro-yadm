# Manjaro yadm bootstrapper

## Standalone bootstrap command for a Manjaro XFCE install:
`sh -c "$(wget -O- https://raw.githubusercontent.com/aa-ko/manjaro-yadm/master/.config/yadm/setup.sh)"`

## Dependencies
- sh
- sudo
- wget
- pacman
- xargs
